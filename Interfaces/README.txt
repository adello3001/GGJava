인터페이스(interface)

1. 객체의 사용 방법을 정의한 타입
2. 객체의 교환성을 높여 주기 위한 역할
3. 객체와 객체 사이의 통신하는 접점 역할
4. 함수의 명세(시그니처)만 정의 
5. 상속을 통한 구현 클래스를 통해서만 객체로 만들 수 있다.
6. 클래스를 규격화(모듈화, 컴포넌트화)

(관련용어)
1. IDL(Interface Definition Language)
2. API(Application Programming Interface)
3. Static Library
4. Dynamic Link Library(DLL)
5. Shared Library(SO)
